# Фичи
## 1. Система напоминаний по email
Как менеджер или исполнитель, я хочу получать напоминания о сроках задачи, а также быть информированным о смене ее статусе и появлении комментариев.
## 2. Отчёт по производительности команды
Как менеджер, я хочу видеть:
- Топ-исполнителей по количеству закрытых задач за период.
- Среднее время выполнения задачи по приоритетам.
- Процент задач, выполненных в срок.

## 3. Kanban-доска для задач
Перетаскивание задач между колонками (TODO → IN_PROGRESS → DONE → BLOCKED) с помощью JavaScript (drag & drop).
При смене статуса автоматически обновляется база и пишется TaskLog.
Можно фильтровать по проекту/спринту.
